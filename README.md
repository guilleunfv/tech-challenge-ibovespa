# Tech Challenge IBOVESPA

Este projeto tem como objetivo prever o fechamento diário do índice IBOVESPA utilizando dados históricos e técnicas de Machine Learning.

## Funcionalidades do Projeto
- Visualização de dados históricos do IBOVESPA.
- Modelo preditivo baseado em séries temporais.
- Aplicação interativa desenvolvida com Streamlit.

## Requisitos
- Python 3.x
- Bibliotecas: `streamlit`, `pandas`, `numpy`, `matplotlib`, `yfinance`.
